This is a CLDR release. For more information on CLDR, see http://cldr.unicode.org/index/downloads

About These Files

  • cldr-common-*.zip contains the common production data (from cldr/common).
  • cldr-keyboards-*.zip contains the keyboard data.
  • cldr-tools-*.zip contains the Java source to the CLDR tools.
  • cldr-tools-*.jar is a standalone .jar file containing the CLDR tools and all needed dependencies.
  • core.zip, keyboards.zip, and tools.zip are symbolic links to the other files, for URL compatibility.
  • cldr-json* - JSON version data, see unicode-org/cldr-json

/hashes subdirectory

This subdirectory contains files which are useful for verifying download integrity.

  • The SHASUM512.txt file contains SHA-512 hashes of the other files.
  • .asc files contain GPG signatures.

Changes from v37

  • The /hashes subdirectory was added.
  • The cldr-tools-*.jar file was added.
  • cldr-tools-*.zip does not contain cldr.jar, use cldr-tools-*.jar instead.
